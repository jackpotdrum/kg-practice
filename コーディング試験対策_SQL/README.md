# コーディング試験対策_SQL

日本企業のコーディング試験（SQL）対策用の学習ログです。  
VSCode + Copilot Chat を使って、**問題演習 → 解答 → 添削 → 記録** を回します。

---

## 目的

- Medium〜Hard相当のSQL問題を安定して解けるようにする
- 試験で頻出のパターンを反復練習する
- 毎回のミスを記録し、再発防止チェックリストで改善する
