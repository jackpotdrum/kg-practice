# 1日3分ラベリング

ITエンジニア向けのラベル付けトレーニングを、VSCode + Copilot Chatで短時間に回すための教材です。  
このディレクトリでは、`prompt.md` の指示に沿って、**出題 → 解答 → 添削 → 記録** を1セットで進めます。

---

## 目的

- 文章や箇条書きから情報を抽出し、要点をラベル化する力を鍛える
- 単純ラベルと分類ラベルを使い分け、説明力と整理力を高める
- 顧客エンジニア・経営者・運用担当者など、相手に応じた抽象度で伝える練習をする

---

## ディレクトリ構成

```text
1日3分ラベリング/
├─ README.md
├─ prompt.md
└─ 学習記録/
	├─ 2026-04-14-001.md
	├─ 2026-04-15-001.md
	└─ ...
```

---

## 使い方（VSCode）

1. VSCodeでこのリポジトリを開く
2. `1日3分ラベリング/prompt.md` をエディタで選択状態にする
3. Copilot Chatで「問題を出して」と入力する
4. 出題された問題に回答する
5. Copilotから模範解答・フィードバックを受ける
6. 記録を残したい場合は「記録して」と入力する

---

## 出題と添削の概要

`prompt.md` では、以下のトレーニング仕様を定義しています。

- 問題形式は「箇条書き型」と「文章型」をバランスよく出題
- 説明相手（顧客エンジニア / 経営者 / 運用担当者）を明示
- 回答後に、模範解答・差分コメント・分類ラベルの思考過程を提示
- 必要に応じて次の問題へ継続

---

## 学習記録ファイル

「記録して」と入力した場合、`学習記録/` 配下に記録ファイルを作成します。

- 保存先: `1日3分ラベリング/学習記録/`
- ファイル名: `yyyy-mm-dd-001.md` 形式
- 同日に複数回実施した場合は `002`, `003`... と3桁連番で採番
- 1ファイルには1回のトレーニングセッション内容を記録

---

## 補足

- 模範解答は唯一解ではなく、実務で使える「良い例」として扱います
- 連続学習する場合は、過去の `学習記録/` を見返して改善点を再確認してください
