# KGプラクティス

VSCode上のCopilot Chatを使って、分野別の練習問題を解くための学習リポジトリです。  
各トレーニングはルート配下のディレクトリごとに分かれており、`prompt.md` の指示に従って進行します。

---

## 学習の進め方（共通）

1. 取り組みたい分野のディレクトリを開く
2. そのディレクトリ内の `prompt.md` を選択状態にする
3. Copilot Chatで「問題を出して」と入力する
4. 回答を入力し、添削・模範解答を受ける
5. セッションを保存したいときは「記録して」と入力する

---

## 共通ディレクトリ構成

各学習ディレクトリは次の構成を持ちます。

```text
{各ディレクトリ}/
├─ README.md
├─ prompt.md
└─ 学習記録/
	 └─ {yyyy}-{mm}-{dd}-{連番(三桁)}.md
```

- `README.md`: その分野の目的、使い方、記録ルール
- `prompt.md`: Copilotへの出題・添削・記録指示
- `学習記録/`: 学習セッションのログ

---

## 収録トレーニング

- `1日3分ラベリング`
	- ラベル付け（単純ラベル / 分類ラベル）を短時間で反復するトレーニング
- `コーディング試験対策_SQL`
	- SQLの実戦問題を解いて採点・添削を受けるトレーニング

---

## 学習記録の命名ルール（共通）

- ファイル名は `yyyy-mm-dd-001.md` 形式
- 同日に複数回実施した場合は `002`, `003`... と3桁連番を採番
- 既存ファイルと重複しない連番を使用

---

## 運用メモ

- 出題内容は毎回新規に生成し、同一テーマの連続を避ける
- 添削では「良かった点」と「改善点」の両方を確認する
- 定期的に `学習記録/` を見返し、再発防止ポイントを更新する
